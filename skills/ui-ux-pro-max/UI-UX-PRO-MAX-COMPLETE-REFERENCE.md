## 67 UI 风格

### 通用风格 (49 种)

#### 1. **Minimalism & Swiss Style**
- **适用场景**: 企业应用、仪表板、文档网站、SaaS 平台、专业工具
- **特点**: 清洁、简单、开阔、功能性、白色空间、高对比度、几何、网格化
- **主要颜色**: 单色、黑色 #000000、白色 #FFFFFF
- **动画**: 微妙悬停 (200-250ms)、流畅过渡、清晰的文字层次、快速加载
- **性能**: ⚡ 优秀
- **无障碍**: ✓ WCAG AAA
- **易用性**: ✓ 高
- **不适用**: 创意作品集、娱乐、娱乐品牌、艺术实验

#### 2. **Neumorphism**
- **适用场景**: 健康/健身应用、冥想平台、健身追踪器、极简交互 UI
- **特点**: 柔和 UI、浮雕/凹陷、光源、微妙深度、圆角 (12-16px)、单色
- **主要颜色**: 浅柔和色: 柔和蓝 #C8E0F4、柔和粉色 #F5E0E8、柔和灰 #E8E8E8
- **动画**: 柔和阴影 (多重: -5px -5px 15px, 5px 5px 15px)、平滑按压 (150ms)
- **性能**: ⚡ 良好
- **无障碍**: ⚠ 低对比度
- **易用性**: ✓ 良好
- **不适用**: 复杂应用、高对比度要求、数据密集型仪表板

#### 3. **Glassmorphism**
- **适用场景**: 现代 SaaS、财务仪表板、高端企业、生活方式应用、模态叠层、导航栏
- **特点**: 磨砂玻璃、透明、模糊背景、分层、充满活力的背景、光源、深度、多层
- **主要颜色**: 半透明白色: rgba(255,255,255,0.1-0.3)
- **次要颜色**: 充满活力: 电蓝 #0080FF、霓虹紫 #8B00FF、生动粉 #FF1493、青色 #20B2AA
- **动画**: 背景模糊 (10-20px)、微妙边框 (1px solid rgba 白 0.2)、光反射、Z 深度
- **性能**: ⚠ 良好
- **无障碍**: ⚠ 确保 4.5:1
- **易用性**: ✓ 良好
- **不适用**: 低对比度背景、关键无障碍、性能受限

#### 4. **Brutalism**
- **适用场景**: 设计作品集、艺术项目、反文化品牌、编辑/媒体网站、技术博客
- **特点**: 原始、未打磨、严苛、高对比度、纯文本、默认字体、可见边框、不对称、反设计
- **主要颜色**: 主色: 红色 #FF0000、蓝色 #0000FF、黄色 #FFFF00、黑色 #000000、白色 #FFFFFF
- **动画**: 无平滑过渡 (即时)、锐角 (0px)、粗体字体 (700+)、可见网格、大块
- **性能**: ⚡ 优秀
- **无障碍**: ✓ WCAG AAA
- **易用性**: ◐ 中等
- **不适用**: 企业环境、保守行业、关键无障碍

#### 5. **3D & Hyperrealism**
- **适用场景**: 游戏、产品展示、沉浸式体验、高端电子商务、建筑可视化、VR/AR
- **特点**: 深度、逼真纹理、3D 模型、空间导航、触觉、拟物化元素、丰富细节、沉浸式
- **主要颜色**: 深海军 #001F3F、森林绿 #228B22、勃艮第红 #800020、金色 #FFD700、银色 #C0C0C0
- **动画**: WebGL/Three.js 3D、逼真阴影 (多层)、物理照明、视差 (3-5 层)、平滑 3D (300-400ms)
- **性能**: ❌ 差
- **无障碍**: ⚠ 不可访问
- **易用性**: ✗ 低
- **不适用**: 低端移动、性能受限、关键无障碍

#### 6. **Vibrant & Block-based**
- **适用场景**: 初创公司、创意机构、游戏、社交媒体、青少年导向、娱乐、消费者
- **特点**: 大胆、充满活力、有趣、块布局、几何形状、高色彩对比度、双色调、现代、充满活力
- **主要颜色**: 霓虹绿 #39FF14、电紫色 #BF00FF、生动粉 #FF1493、亮青色 #00FFFF、日出 #FFAA00
- **动画**: 大型部分 (48px+ 间距)、动画图案、大胆悬停 (色彩移位)、滚动捕捉、大字体 (32px+)、200-300ms
- **性能**: ⚡ 良好
- **无障碍**: ◐ 确保 WCAG
- **易用性**: ✓ 高
- **不适用**: 金融机构、医疗、正式商务、政府、保守、老年

#### 7. **Dark Mode (OLED)**
- **适用场景**: 夜间模式应用、编码平台、娱乐、眼睛疲劳预防、OLED 设备、低光
- **特点**: 深色主题、低光、高对比度、深黑色、午夜蓝、眼睛友好、OLED、夜间模式、节能
- **主要颜色**: 深黑色 #000000、深灰色 #121212、午夜蓝 #0A0E27
- **次要颜色**: 充满活力的口音: 霓虹绿 #39FF14、电蓝 #0080FF、金色 #FFD700、等离子紫 #BF00FF
- **动画**: 最小辉光 (text-shadow: 0 0 10px)、深至浅过渡、低白色发射、高可读性、可见焦点
- **性能**: ⚡ 优秀
- **无障碍**: ✓ WCAG AAA
- **易用性**: ✓ 高
- **不适用**: 印刷优先内容、高亮度户外

#### 8. **Accessible & Ethical**
- **适用场景**: 政府、医疗、教育、包容性产品、广泛受众、法律合规、公共
- **特点**: 高对比度、大文本 (16px+)、键盘导航、屏幕阅读器友好、WCAG 符合、焦点状态、语义
- **主要颜色**: WCAG AA/AAA (4.5:1 最小)、简单主色、清晰次色、高亮度 (7:1+)
- **动画**: 清晰焦点环 (3-4px)、ARIA 标签、跳过链接、响应式设计、减少运动、44x44px 触摸目标
- **性能**: ⚡ 优秀
- **无障碍**: ✓ WCAG AAA
- **易用性**: ✓ 高
- **不适用**: 无 - 无障碍是普遍的

#### 9. **Claymorphism**
- **适用场景**: 教育应用、儿童应用、SaaS 平台、创意工具、有趣导向、入门、休闲游戏
- **特点**: 柔和 3D、厚重、有趣、玩具般、泡泡、厚边框 (3-4px)、双阴影、圆角 (16-24px)
- **主要颜色**: 柔和: 柔和桃 #FDBCB4、婴儿蓝 #ADD8E6、薄荷 #98FF98、淡紫 #E6E6FA、浅 BG
- **动画**: 内+外阴影 (微妙、无硬线)、柔软按压 (200ms ease-out)、毛绒元素、平滑过渡
- **性能**: ⚡ 良好
- **无障碍**: ⚠ 确保 4.5:1
- **易用性**: ✓ 高
- **不适用**: 正式企业、专业服务、数据关键、严肃/医疗

#### 10. **Aurora UI**
- **适用场景**: 现代 SaaS、创意机构、品牌、音乐平台、生活方式、高级产品、英雄部分
- **特点**: 充满活力的渐变、平滑混合、北极光效果、网格渐变、发光、大气、抽象
- **主要颜色**: 互补: 蓝-橙、紫-黄、电蓝 #0080FF、洋红 #FF1493、青色 #00FFFF
- **动画**: 大型流动 CSS/SVG 渐变、微妙 8-12s 动画、通过颜色分层实现深度、平滑变形
- **性能**: ⚠ 良好
- **无障碍**: ⚠ 文本对比度
- **易用性**: ✓ 良好
- **不适用**: 数据密集型仪表板、关键无障碍、内容繁重可能分散注意力

### 着陆页风格 (8 种)

#### 1. **Hero-Centric Design**
- **特点**: 强大视觉标识、大型英雄镜像、立即 CTA
- **最佳用途**: 产品具有强大视觉身份

#### 2. **Conversion-Optimized**
- **特点**: 清晰价值主张、多个 CTA、社会证明、快速加载
- **最佳用途**: 潜在客户开发、销售页面

#### 3. **Feature-Rich Showcase**
- **特点**: 多个功能部分、屏幕截图、用例、比较
- **最佳用途**: SaaS、复杂产品

#### 4. **Minimal & Direct**
- **特点**: 一个清晰消息、无分散、快速操作
- **最佳用途**: 简单产品、应用

#### 5. **Social Proof-Focused**
- **特点**: 推荐、案例研究、用户头像、评分
- **最佳用途**: 服务、B2C 产品

#### 6. **Interactive Product Demo**
- **特点**: 可播放演示、屏幕录制、交互式元素
- **最佳用途**: 软件、工具

#### 7. **Trust & Authority**
- **特点**: 证书、专业团队、详细背景、长篇幅内容
- **最佳用途**: B2B、企业、咨询

#### 8. **Storytelling-Driven**
- **特点**: 品牌故事、个人联系、视觉叙述
- **最佳用途**: 品牌、机构、非营利

### 优雅系列

#### 1. **Classic Elegant** - Playfair Display + Inter
- **心情**: 优雅、豪华、精致、永恒、高级、编辑
- **最佳用途**: 奢侈品牌、时尚、水疗、美容、编辑、杂志、高端电商
- **Google Fonts URL**: https://fonts.google.com/share?selection.family=Inter:wght@300;400;500;600;700|Playfair+Display:wght@400;500;600;700

- **Tailwind配置**: `fontFamily: { serif: ['Playfair Display', 'serif'], sans: ['Inter', 'sans-serif'] }`

## 使用指南

### 第一步：识别产品类型
查看上面的**行业规则**找到你的产品类型。

### 第二步：选择风格
- 查找推荐的**样式**
- 审查**次要样式**
- 考虑**性能/无障碍**影响

### 第三步：选择颜色
- 查找对应的**色彩方案**
- 验证**对比度** (WCAG AA 最小值)
- 测试**浅色和深色**模式

### 第四步：选择字体
- 查找对应的**字体配对**
- 验证 **Google Fonts 导入**
- 设置 **Tailwind 配置**

### 第五步：选择技术栈
- 根据你的需求选择**栈**
- 验证**性能目标**
- 检查**开发者体验**

## 关键提示

✅ **做什么**:
- 组合最多 3 种风格
- 对按钮使用对比色
- 测试真实内容
- 优先考虑无障碍性
- 在移动设备上测试
- 验证加载时间 (< 2s)

❌ **不要做什么**:
- 混合太多风格 (不连贯)
- 仅使用颜色来区分 (无障碍问题)
- 忽视 WCAG 对比要求
- 添加不必要的动画
- 忘记焦点状态
- 在低带宽上测试
