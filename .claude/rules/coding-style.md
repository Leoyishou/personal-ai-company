# 编码风格

## 文档写作

- 一级标题：中文数字（一、二、三）
- 二级标题：阿拉伯数字加小数点（1.1、1.2、2.1）

## UI/网页

自建网页/HTML 页面优先使用 Light 风格（白底、浅色系），除非用户明确要求 dark 主题。

## 问题分析

面对问题时，先思考：这是什么经典问题的变体？有什么已知解法？

## B站投稿

账号：转了码的刘公子

图文内容发 B站 需先用 ffmpeg 合成视频（图片轮播 + TTS 配音 + 字幕烧录），再用 biliup 上传。

## ASR 优先级

1. 火山引擎（需配置环境变量）
2. faster-whisper（本地备选）

## 社交媒体文案

**禁止使用的 emoji：**
- 数字圆圈类：1️⃣ 2️⃣ 3️⃣ 等
- 其他廉价感 emoji

**推荐的列表格式：**
- 用纯文字序号：1. 2. 3. 或 一、二、三
- 或用短横线：-
- 保持简洁干净的文字风格
