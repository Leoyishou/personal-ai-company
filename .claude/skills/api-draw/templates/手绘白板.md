# 手绘白板风格

**适用场景：** 流程图、知识框架、方法论、系统架构、AI/科技话题、商业概念可视化

**设计特点：**
- 类似 Excalidraw 的手绘草图美学
- 极简线条艺术，粗黑马克笔线条
- 蓝色高亮点缀作为强调色
- 简单涂鸦图标 + 流程图结构
- 纯白背景，干净专业
- 箭头连接各元素，展示数据流/逻辑流

## Prompt 模板

```
生成一张小红书风格封面图，3:4竖版比例，
手绘白板草图风格，视觉笔记美学，
极简线条艺术，粗黑马克笔线条，蓝色高亮点缀，
简单涂鸦图标，流程图结构，箭头连接各元素，
纯白背景，干净专业，商业概念可视化。

顶部标题：「[主标题]」用粗体手写风格，蓝色下划线强调。

内容布局（从左到右/从上到下的流程）：
- 第一部分「[模块名]」：[图标描述1]、[图标描述2]
- 第二部分「[模块名]」：[图标描述1]、[图标描述2]
- 第三部分「[模块名]」：[图标描述1]、[图标描述2]

用粗箭头连接各部分，展示流程走向。
风格：Excalidraw 手绘美学，无阴影，无渐变，无3D效果，略带不规则的草图线条。
```

## 选题适配指南

**核心原则：** 这种涂鸦风格的图标必须是「抽象符号」，而非「具体实物外形」。

**适合的选题类型：**

| 选题类型 | 示例 | 图标可以用 |
|----------|------|-----------|
| 品牌/平台推荐 | 5个免费AI课程 | 品牌logo天然是简化符号 |
| 工具/App推荐 | 5个学车App | 用功能符号代替精确logo |
| 流程/方法论 | 高效学习5步法 | 书 笔 大脑 灯泡 |
| 抽象概念 | 打工人早起5件事 | 闹钟 咖啡 手机 |

**不适合的选题类型：**

| 选题类型 | 示例 | 问题 |
|----------|------|------|
| 具体产品对比 | 保时捷5大经典车型 | 车型需要外形细节才能区分 |
| 需要精确外形的实物 | 5款经典耳机推荐 | 耳机形状是核心 |
| 人物介绍 | 5位科技大佬 | 人脸无法简化成涂鸦 |

**选题公式：**
```
好选题 = 抽象概念/品牌/工具 + 功能符号图标
坏选题 = 具体实物 + 需要外形细节才能辨认
```

**图标替代技巧：**

当品牌logo不够简洁时，用「功能符号」替代「精确logo」：

| 原始需求 | 替代图标 | 理由 |
|----------|----------|------|
| 驾考宝典logo | 蓝盾牌+勾 | 传达"题库/保障"感 |
| 驾校一点通logo | 绿色方向盘 | 传达"练车"功能 |
| 车轮驾考logo | 橙色轮子 | 品牌名直接可视化 |
| 交管12123 | 星星证书 | 传达"官方"感 |

**Prompt 强化技巧（让图标更涂鸦）：**

```
VERY simple doodle, NOT realistic
childlike simple sketch, crude imperfect lines
like drawn with marker on whiteboard
wobbly hand-drawn style, like stick figures
```
