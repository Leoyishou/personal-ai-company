# 板书插画风格

**适用场景：** 财经理财、知识科普、教育课程、入门教学、干货分享、技术教程

**设计特点：**
- 粗毛笔手写中文标题，**飞白效果**（笔画断裂感、墨迹飞溅）
- **白板结构**：深灰色粗边框 + **白色内部背景**（不是深色填充！）
- 白板底部有**两个支脚**（像真实的演示白板架）
- 白板内展示手绘图表或列表（饼图、曲线图、标签列表等）
- 金黄色外套的卡通人物，举手指向白板的教学姿势
- 散落的主题相关装饰（金币、星星、铅笔、代码符号等）
- 纯白背景，干净明亮
- 整体活泼但专业，知识课堂感

**关键细节（必须遵守）：**
1. 白板是 **边框深灰 + 内部白色**，不是全深色
2. 标题要有 **飞白效果**（flying white, broken brush strokes, ink splatter）
3. 白板要有 **支脚**（leg stands at bottom）

## 配色方案

| 颜色 | Hex | 用途 |
|------|-----|------|
| 金黄色（主色调） | #F5A623 | 人物衣服、金币、图表高亮 |
| 黑色 | #1A1A1A | 标题、边框、线条 |
| 深灰 | #4A4A4A | 白板框架边框 |
| 白色 | #FFFFFF | 整体背景、白板内部、人物裤子 |
| 薄荷绿 | #A8D5BA | 标签/徽章背景色 |
| 点缀色 | 浅蓝/浅橙 | 饼图分区、图表元素 |

## 与手绘白板风格的区别

| 维度 | 手绘白板 | 板书插画 |
|------|---------|----------|
| 人物 | 无/简笔火柴人 | 完整精致卡通人物 |
| 线条 | 粗糙涂鸦感 | 干净插画线条 |
| 配色 | 黑+蓝点缀 | 黑+金黄主色调 |
| 装饰 | 极简箭头 | 立体金币、星星、齿轮等 |
| 氛围 | 极客/专业 | 活泼/教育 |
| 适用 | 流程图、架构图 | 知识科普、入门教学、技术教程 |
| **白板** | 无边框/简单框 | **深灰边框+白色内部+支脚** |
| **标题** | 普通手写 | **飞白效果、墨迹飞溅** |
